## 前言 为什么是pm2.5 ？

这本书的名字，暂且定为pm2.5,因为2.5比2大，又比3小。可以认为2.5是一个版本号，位于2和3之间。这样可以理解为，做2.5版本的产品经理，比1.0强，比2.0强，但是还没有到3,为什么？因为3.0的产品经理应该是乔布斯和张小龙，所以我们要保持一种畏惧，保持一种自我激励。3.0是目标，但我们在目前的努力能做好2.5即可。
